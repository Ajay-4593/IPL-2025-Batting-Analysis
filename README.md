# IPL 2025 Batting Analysis

## 📊 Overview
This project analyzes IPL 2025 batting data using Python.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 🔍 Key Insights
- Identified aggressive players using strike rate
- Compared team performance based on sixes
- Analyzed consistency using number of 50s

## 📈 Visualizations
- Top 5 six hitters
- Top 5 strike rates
- Team-wise six comparison

## 🚀 Conclusion
This project demonstrates how data analysis can be used to extract meaningful insights from sports data.
